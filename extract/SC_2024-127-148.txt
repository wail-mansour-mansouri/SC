Les modèles intégrés sont très présents dans les séries économiques, par

Tests de racine unitaire : les tests de Dickey-Fuller
Le test de Dickey & Fuller propose de tester 3 situations
1 H0 : Xt est une marche aléatoire (RW) avec tendance et dérivé
Xt −ϕXt−1 = α + βt + at H0 : β = 0
2 H0 : Xt est une marche aléatoire (RW) avec dérivé
Xt −ϕXt−1 = α + at H0 : α = 0
3 H0 : Xt est une marche aléatoire (RW)
Xt −ϕXt−1 = at H0 : ϕ = 1
L'hypothèse alternative H1 : ϕ < 1 correspond au cas d'un modèle AR(1)

Processus ARIMA

intégration d'ordre d, Xt ∼I(d).
Yt = (1 −B)dXt
Si le processus sortant Yt est un ARMA(p,q), alors le processus entrant Xt
est un ARIMA(p,d,q). Un ARIMA(p,d,q) s'écrit :

===== PAGE 11 =====
Processus ARIMA
Dans la plupart des chroniques économiques, les tests DF indiquent la
présence d'une seule racine unitaire et les chroniques sont dites intégrées
d'ordre 1, I(1).
Si la chronique est stationnaire, on dit qu'il est I(0). en d'autres termes, un
ARIMA(p,0,q)=ARMA(p,q)
Parmi les processus ARIMA non stationnaires, on peut citer le processus
ARIMA(0,1,0) : (1 −B)Xt = at ⇔xt = xt−1 + at, qui est une marche
aléatoire (RW)(processus DS le plus simple).
